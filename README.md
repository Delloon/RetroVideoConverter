# RetroVideoConverter

Простая программа для конвертации видео в форматы, подходящие для старых устройств.

Проект появился из практической необходимости — **подготовить видео для воспроизведения на старом кнопочном мобильном телефоне**.

Приложение автоматически уменьшает разрешение, подбирает кодеки и выполняет конвертацию через встроенный **FFmpeg**. Проект сделан как небольшой инструмент для личного использования и экспериментов с конвертацией видео для старых устройств.

## Особенности

- Не требует установки FFmpeg — он **встроен в программу**
- Простой интерфейс
- Поддержка нескольких файлов
- Автоматическое определение доступных кодеков
- Сохранение пропорций видео
- Быстрая пакетная конвертация

## Поддерживаемые форматы

### Входные
- MP4
- AVI
- MKV
- MOV
- WMV
- FLV
- WEBM

### Выходные
- **3GP**
- **MP4**
- **AVI**

## Поддерживаемые кодеки

В зависимости от возможностей компьютера автоматически доступны:

**CPU**
- H.263
- MPEG-4
- H.264 (x264)

**GPU (если поддерживается системой)**
- NVIDIA NVENC
- Intel QSV
- AMD AMF

## Поддерживаемые разрешения

- 96p (128×96)
- 144p (176×144)
- 240p (320×240)
- 288p (352×288)
- 360p (480×360)
- 480p (640×480)

## Основные возможности

- пакетная обработка видео
- выбор кодека
- выбор контейнера
- автоматическое определение доступных кодеков
- сохранение пропорций
- лог процесса конвертации
